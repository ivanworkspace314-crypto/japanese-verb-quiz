# Japanese Quiz App

This project is a Japanese Quiz App built with Next.js.

## Purpose

1. Practice Japanese through quiz-style questions.
2. Practice transitive and intransitive verb pairs.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js
- React
- JavaScript
